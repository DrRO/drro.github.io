Zakat
Calculation App


This application is an electronic calculator for zakat
with the ability to make a plan for your zakat.

Application features:

  1. This is native app specified for Android system using java.
  2. I use function to calculate the money amount.
  3. It contain professional Setting Pannel.
  4. I use API which get gold price for selected country.
  5. The app contain todo list to plan your Zakat.
  6. Notification system to alert you the time of your Zakat.
  7. The application contain types of Zakat.
  8. Users Can select their country's currency.
  9. It support English & Arabic Language.
